This is an unmodified copy of Lua 5.2.1, with the additions of some
project files for Visual Studio 2010 and 2012.

The project files can be found in ./VS2010/ and ./VS2012/

Compiled output is placed in ./bin/

When using in your own projects, will you will probably want to copy
the header files, ./src/*.h and ./src/lua.hpp into your projects.

Distribution put together by Sean Middleditch, website at
http://seanmiddleditch.com/.  The project files are under the
normal Lua license, no warranty, do whatever you want with them.
